    利用 GitHub Pages 和 Hugo 轻松搭建个人博客

    转自：https://zhuanlan.zhihu.com/p/57361697 1. 新建自己的github仓库 2. 安装Hugo 参考链接：https://gohugo.io/installation/ 我这里使用Scoop安装： Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression 3. 新建一个Hugo网站 在你想存放Hugo网站文件夹的目录中打开终端，执行以下指令： hugo new site lilian-blog # "lilian-blog" 是我的网站文件夹名。 4. 选择一个主题并克隆到本地 4.1 打开 Hugo Themes 页面，选择一个你喜欢的主题。 Hugo Themes: https://themes.gohugo.io 4.2 将所选主题克隆至本地目录。 在网站根目录执行以下命令下载主题并初始化为 Git 子模块： git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1 git init git submodule--helper add https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod --depth=1 4.3 编辑配置文件 打开根目录下的配置文件hugo.toml，在文件中添加一行： theme = "PaperMod" # 注意，theme 配置项指的是所选主题的名称，必须与所选主题被克隆至的目录名相同 5. 新建一篇文章 在根目录中执行： hugo new posts/my-first-post.md 这时会在content目录下生成posts/my-first-post.md，编辑新建的文章，添加自己需要的内容。 6. 本地预览网页效果 6.1 启动 Hugo server hugo server -D 6.2 在网页中预览 网页中访问： http://localhost:1313/，查看效果。 ...
